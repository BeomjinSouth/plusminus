# QUESTION_BANK — 점프 계산 문제 은행

작성일: 2026-03-23  
세트당 문항 수: 3문항

---

## 1. 운영 원칙

- 점프 계산 모델 하나만 사용한다.
- 난이도는 `쉬움 / 보통 / 어려움` 3세트다.
- 모든 문항은 고정형 문제 은행으로 운영한다.
- 공통분모는 12 이하를 원칙으로 한다.
- 수직선 눈금은 문제별 `gridDenominator` 로 계산한다.

## 5. 모델 사용 방식

- 모든 문항은 점프 계산으로만 제공한다.
- 하/중 난이도는 학생이 빠르게 감을 잡도록 짧게 진행한다.
- 상 난이도는 반드시 `끊기 → 부호 정리 → 점프 → 답` 흐름을 모두 사용한다.

---

## 6. 회귀 우선 문항

- `-(+3)-(-5)+7`
- `(+2/3)-(-5/6)-(+1/2)`
- `(+2.4)-(-1.1)-(+0.5)`
