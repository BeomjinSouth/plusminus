# PlusMinus Lab

유리수의 덧셈과 뺄셈을 **점프 계산** 하나로 연습하는 Next.js 웹앱입니다.

## 현재 범위

- 모델: `rabbit-sign-parser`만 사용
- 난이도: `low`, `medium`, `high`
- 세트당 문항 수: 7문항
- 로그 저장: Google Sheets append

## 주요 흐름

1. 로그인
2. 난이도 선택
3. 식 끊기
4. 부호 정리
5. 수직선 점프
6. 최종 값 입력

## 개발

```bash
npm install
npm run test
npm run build
npm run dev
```
