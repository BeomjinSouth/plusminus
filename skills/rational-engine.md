# Skill: rational-engine

## 목적
유리수의 덧셈·뺄셈을 부동소수점 오차 없이 정확하게 처리한다.

## 반드시 지켜야 할 것
- 내부 계산은 분자/분모 또는 Fraction 객체 사용
- 입력은 정수, 분수, 유한소수 모두 허용
- 출력은 문제 의도에 맞게 문자열 포맷팅
- `number` 덧셈으로 정답 판정 금지

## 체크리스트
1. 문자열 입력 정규화
2. 부호 분리
3. 분수/소수/정수 파싱
4. 기약분수화
5. 누적합 생성
6. 수직선 눈금 단위 계산
7. 표시 문자열 변환

## 회귀 테스트
- 2.4 - (-1.1) - 0.5 = 3
- 2/3 + 5/6 - 1/2 = 1
- -3/4 + 5/4 - 1/2 = 0
